Rows & Grids

Columns can be further divided up by the use of rows and grids to a maximum of 12 grid columns. The maximum number of grids that can be utilized depends on the number of columns you have.

Rows

This is 1 of 2 grids inside of a row.
This is 2 of 2 grids inside a row.

Rows are used to contain grid columns. Each row can be divided up into 12 grid columns. Rows span the full percentage width of its containing element. Rows are created by a div.int-row.

                            Grids are used to divide up a row by up to 12 columns. The grid class is in the format .int-grid$ ($ is a number from 1-12), and can be applied to many components to span the numbers of columns desired within the row i.e. .int-grid12 spans all 12 columns whereas .int-grid3 would span 3 columns out of 12.  The 12 columns can be made up by a combination of different components with different grid classes to make up the 12 i.e. .int-grid5 and .int-grid7; 5+7=12.
                        

                        

                            The background colour in the above example is for visual reference only and will not appear in standard application styles.

1/3rd and 2/3rd columns are defined as .int-col1/3rd and .int-col2/3rd. .int-col1/3rd has been designed to be used as a sidebar, as an example for a navigation or an aside. The maximum grid combination for .int-col1/3 is up to .int-grid4. The maximum grid combination for .int-col2/3rd is up to .int-grid8. See Columns for more details.

                        

                            It is not recommended to populate .int-col1/3rd with a form
