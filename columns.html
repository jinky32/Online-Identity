If your page inherits from the two column layout:

                        
@ {
    Layout = "~/Views/Shared/_TwoColumnLayout.cshtml";
}

@section col_1of2 {
    <!--Left column content goes here-->
}

@section col_2of2 {
    <!--Right column content goes here-->
}


        
If your page uses the two column layout inline:

        
<div class="int-wrap">
    @using (Html.BeginTwoColLeft()) 
    {
        <!--Left column content goes here-->
    }

    @using (Html.BeginTwoColRight()) 
    {
        <!--Right column content goes here-->
    }
</div>
                		

                    

                    

                        
To create a 2 column layout, create a div.int-wrap with child elements div.int-col1of2 and div.int-col2of2.

If your page inherits from the three column layout:

                        
@ {
    Layout = "~/Views/Shared/_ThreeColumnLayout.cshtml";
}

@section col_1of3 {
    <!--Left column content goes here-->
}

@section col_2of3 {
    <!--Centre column content goes here-->
}

@section col_3of3 {
    <!--Right column content goes here-->
}
                		

                        
If your page uses the three column layout inline:

                        
<div class="int-wrap">
    @using (Html.BeginThreeColLeft()) 
    {
        <!--Left column content goes here-->
    }

    @using (Html.BeginThreeColCentre()) 
    {
        <!--Centre column content goes here-->
    }

    @using (Html.BeginThreeColRight()) 
    {
        <!--Right column content goes here-->
    }
</div>
                		

                    

                    

                        
To create a 3 column layout, create a div.int-wrap with child elements div.int-col1of3, div.int-col2of3 and div.int-col3of3.
