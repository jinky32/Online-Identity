@using (Html.BeginTable("Basic Table", false, false))
{
    using (Html.BeginTableHead())
    {
        using (Html.BeginTableRow())
        {
            @Html.TableHeadCell("Last Name")
            @Html.TableHeadCell("First Name")
            @Html.TableHeadCell("Email")
            @Html.TableHeadCell("Web Site")
        }
    }
    using (Html.BeginTableBody())
    {
        foreach (var row in Model.TableRows)
        {
            using (Html.BeginTableRow())
            {
                @Html.TableCell(row.FirstName)
                @Html.TableCell(row.LastName)
                @Html.TableCell(row.Email)
                @Html.TableCell(row.Website.AbsoluteUri)
            }
        }
    }
}

The basic table is a standard html table. This includes div.int-tableFeature which adds block style to the table as well as a horizontal scroll bar (auto). The caption is optional and adds the grey bar at the top of the table header.

Text alignment can be cutomised by adding int-center and int-right class names to th and tds to align text values within table cells.

To set the coloumn width, create a colgroup tag with child col tags corresponding to the number of columns you have. The table column width is set by changing the class of the col tag:

  • col.int-c10: 10% width
  • col.int-c20: 20% width
  • col.int-c30: 30% width
  • col.int-c40: 40% width
  • col.int-c50: 50% width
  • col.int-c60: 60% width
  • col.int-c70: 70% width
  • col.int-c80: 80% width
  • col.int-c90: 90% width

@using (Html.BeginTable("Expanding Nested Table", false, true))
{
    using (Html.BeginTableHead())
    {
        using (Html.BeginTableRow())
        {
            @Html.TableHeadCell("Last Name")
            @Html.TableHeadCell("First Name")
            @Html.TableHeadCell("Email")
            @Html.TableHeadCell("Web Site")
        }
    }
    using (Html.BeginTableBody())
    {
        using (Html.BeginTableRow(true))
        {
            @Html.TableCell("Crystal")
            @Html.TableCell("Robert", 3)
        }
        using (Html.BeginNestedTableRow(4))
        {
            using (Html.BeginChildTable())
            {
                using (Html.BeginTableHead())
                {
                    using (Html.BeginTableRow())
                    {
                        @Html.TableHeadCell("Last Name")
                        @Html.TableHeadCell("First Name")
                    }
                }
                using (Html.BeginTableBody())
                {
                    foreach (var row in Model.TableRows)
                    {
                        using (Html.BeginTableRow())
                        {
                            @Html.TableCell(row.FirstName)
                            @Html.TableCell(row.LastName)
                        }
                    }
                }
            }
        }
        foreach (var row in Model.TableRows)
        {
            using (Html.BeginTableRow())
            {
                @Html.TableCell(row.FirstName)
                @Html.TableCell(row.LastName)
                @Html.TableCell(row.Email)
                @Html.TableCell(row.Website.AbsoluteUri)
            }
        }
    }
}

An expanding nested table is comprised of the parent table and child tables. Each child table requires two rows to function correctly. One row is allocated as the trigger row to expand the child table. The other row is the row containing the child table and is hidden until triggered.

The table row that contains the child table has a .int-childTable. The table is contained within a td that spans the whole row.

Only basic HTML markup will be valid for the child table. Caption and div.int-tableFeature should not be added.

As there may be many child tables within the parent table, there is an optional button which toggles all the child tables to expand. The HTML markup for this button a.int-button.int-expandCollapseAll[href="#"]
