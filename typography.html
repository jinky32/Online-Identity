Description here.

    

    
Title 3 (i.e. Product Name)

    

        
Description here.

    



                    

                    

                        
Razor

                        
HTML

                        
Supporting Text

                        
Accessibility

                    

                    

                    
@using (Html.BeginDefinitionList("Definition List"))
{
    @HTML.DefinitionListItem("Title 1 (i.e. Product Name)", "Description 1 here.")
    @HTML.DefinitionListItem("Title 2 (i.e. Product Name)", "Description 2 here.")
    @HTML.DefinitionListItem("Title 3 (i.e. Product Name)", "Description 3 here.")
}
                

                    

                        

                            Definition lists are constructed using dl (definition list) which are used for grouping the list, dt (definition title) and dd (definition description). dd's are used for grouping descriptions to dt's which allows the placement of nested HTML elements. 
                        

                    

                    

                        

                            dl's must include a descriptive title (much like how a legend is used to describe a fieldset within a form).
