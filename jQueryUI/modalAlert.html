Modal Alert

Show Modal

This is a modal alert

Use me for alerts such as confirmation dialog.

Razor
HTML
JS
Supporting Text
Accessibility
Changelog

                                

                                    
The modal contains a container div for alerts, with an id in the format of modalId-alert, where modalId is the id passed in to the helper (e.g. alertModal-alert). You can attach alerts within the modal by calling the jQuery alert plugin and using the modalId-alert as a selector.

                                    
@HTML.ModalAlert("alertModal", "This is a modal alert", @<text><p>Are you sure you want to close this modal?</p></text>)
                            		

                                

                                

                                

                                    
Modal Alert should only be used for alert messages where user response is required i.e. Confirmation - 'Are you sure you want to delete'.

                                    
Modal alerts use a basic modal layout. The modal alert uses div#custom_id, with a title attribute to display the type of alert. The content inside the modal is within a div.int-dialogWrap.

                                    
JS

                                    
Select the custom id of the main div of the modal in the dialog function and define required function options:

                                    
    
                                        
  • autoOpen: usually set to false as the open is usually triggered by a user event.
    • 
                                        
  • height: sets the height
    • 
                                        
  • width: sets the width
    • 
                                        
  • modal: set to true. A false value will return a dialogue box instead of a modal.
    • 
                                        
  • resizable: allows the modal to be resized.   
    • 
                                        
  • draggable: allows the modal to be dragged to different positions across the page.
    • 
                                        
  • buttons: sets the default buttons and functionality of the buttons.
    • 
                                    

                                    
The modal trigger event is defined at the end of the JS. In this example we have used a button click event to trigger the modal to open (using the selector #opener).
